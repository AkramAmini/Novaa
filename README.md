#  NOVA — Cosmic Experience Landing Page

A modern, futuristic landing page built with pure HTML & CSS.  
This project showcases advanced UI/UX design with a cosmic theme.

---

##  Features

- Glassmorphism design with blur and transparency effects
- Dark / Light mode toggle
- Animated plasma sphere with floating motion
- Horizontal scrolling feature section
- Interactive hover effects and smooth transitions
- Fully responsive for all screen sizes

---

## 🛠 Technologies Used

- HTML5
- CSS3
- Flexbox & CSS Grid
- CSS Animations & Transitions
- Font Awesome & Google Fonts

---

##  What I Learned

- How to create a complete landing page from scratch
- How to use modern CSS techniques like Glassmorphism
- How to make a website fully responsive
- How to manage design systems and visual consistency
